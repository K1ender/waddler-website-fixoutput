Create a `index.ts` file in the `src` directory and initialize the connection:

<CodeTabs items={["bun sql", "bun sql with config"]}>
```typescript copy
import 'dotenv/config';
import { waddler } from 'waddler/bun-sql';

const sql = waddler(process.env.DATABASE_URL!);
```
```typescript copy
import 'dotenv/config';
import { waddler } from 'waddler/bun-sql';

// You can specify any property from the bun sql connection options
const sql = waddler({ connection: { url: process.env.DATABASE_URL! }});
```
</CodeTabs>

If you need to provide your existing driver:
```typescript copy
import 'dotenv/config';
import { waddler } from 'waddler/bun-sql';
import { SQL } from 'bun';

const client = new SQL(process.env.DATABASE_URL!);
await client.connect();
const sql = waddler({ client });
```
