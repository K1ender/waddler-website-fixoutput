Chunked features is not implemented in `planetscale` and `tidb` drivers
</Callout>

`.chunked()` lets you stream query result rows in chunks:
```ts
const result = sql`select * from users`.chunked(2);

for await (const chunk of result) {
  console.log(chunk);
}
```
