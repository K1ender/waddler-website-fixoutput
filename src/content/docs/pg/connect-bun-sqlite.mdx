# Waddler \<\> Bun SQLite

<Prerequisites>
- Database [connection basics](/docs/connect-overview) with Waddler
- Bun - [website](https://bun.sh/docs)
- Bun SQLite driver - [docs](https://bun.sh/docs/api/sqlite)
</Prerequisites>

According to the **[official website](https://bun.sh/)**, Bun is a fast all-in-one JavaScript runtime. 

Waddler natively supports **[`bun:sqlite`](https://bun.sh/docs/api/sqlite)** module and it's crazy fast ðŸš€  

We embrace SQL dialects and dialect specific drivers and syntax and mirror 
SQLite-like `all`, `run` query methods syntax. 

#### Step 1 - Install packages
<Npm>
waddler
</Npm>

#### Step 2 - Initialize the driver and make a query
```typescript copy
import { waddler } from 'waddler/bun-sqlite';

const sql = waddler();

const result = await sql`select 1;`.all();
```

If you need to provide your existing driver:
```typescript copy
import { waddler } from 'waddler/bun-sqlite';
import { Database } from 'bun:sqlite';

const sqlite = new Database('sqlite.db');
const sql = waddler({ client: sqlite });

const result = await sql`select 1;`.all();
```
