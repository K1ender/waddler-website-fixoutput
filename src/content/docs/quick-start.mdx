# Quick start with Waddler
<Prerequisites>
- DuckDB database and NodeJS driver - [read here](https://duckdb.org/docs/api/nodejs/overview)
</Prerequisites>
<br/>

**Step 1** - install dependencies:
<Npm>
waddler duckdb
</Npm>


**Step 2** - initialise Waddler instance:
```ts
import { waddler } from "waddler";

const sql = waddler({ dbUrl: "local.db", min: 1, max: 1 })
```

**Step 3** - run your first query:
```ts
const result = await sql`select 1`;
```
