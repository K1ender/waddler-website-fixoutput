# Waddler SQL identifier
<Prerequisites>
- Quick start with Waddler - [read here](/docs/sqlite/get-started/sqlite-new)
- Waddler SQL template - [read here](/docs/sqlite/sql-template)
</Prerequisites>
<br/>

`sql.identifier` lets you conveniently pass identifiers to SQL query with automatic escaping:
<Section>
```ts
await sql`select * from ${sql.identifier("users")}`;
await sql`select ${sql.identifier("id")} from ${sql.identifier("users")}`;
await sql`select ${sql.identifier(["id", "name", "email"])} from ${sql.identifier("users")}`;
```
```sql
select * from "users";
select "id" from "users";
select "id", "name", "email" from "users";
```
</Section>

You can explicitely provide `table` and `column` to the query:
<CodeTabs items={["tables", "columns", "columns and tables"]}>
<Section>
```ts
const users = sql.identifier({ table: "users" });
await sql`select * from ${users}`;
```
```sql
select * from "users";
```
</Section>
<Section>
```ts
const id = sql.identifier({ table: "users", column: "id" });
await sql`select ${id} from "users"`;

const multipleColumns = sql.identifier([
	{ table: "users", column: "id" }
	{ table: "users", column: "name" }
	{ table: "users", column: "email" }
]);
await sql`select ${multipleColumns} from "users"`;
```
```sql
select "users"."id" from "users";
select "users"."id", "users"."name", "users"."email" from "users";
```
</Section>
<Section>
```ts
const table = sql.identifier({ table: "users" })
const columns = sql.identifier([
	{ ...table, column: "id" }
	{ ...table, column: "name" }
	{ ...table, column: "email" }
]);
await sql`select ${columns} from ${table}`;
```
```sql
select "users"."id", "users"."name", "users"."email" from "users";
```
</Section>
</CodeTabs>

SQL identifier has support for `as`:
<Section>
```ts
const table = sql.identifier({  table: "users", as: "u" })
const id = sql.identifier({ table: "u", column: "id", as: "userId" });
await sql`select ${columns} from ${table}`;
```
```sql
select "u"."id" as "userId" from "users" "u";
```
</Section>
