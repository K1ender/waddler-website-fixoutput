`sql.unsafe` lets you use a client itself by providing sql string and params

<Section>
```ts
const response = await sql.unsafe('select * from users where id = {param1:String}', { "param1": 10 });
```
```sql
select * from users where id = {param1:String}; -- params: { "param1": 10 }
```
