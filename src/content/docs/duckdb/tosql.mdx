# Waddler toSQL
<Prerequisites>
- Quick start with Waddler - [read here](/docs/duckdb/get-started/duckdb-neo-new)
- Waddler SQL template - [read here](/docs/duckdb/sql-template)
</Prerequisites>
<br/>

`.toSQL()` prepares and prints you SQL query as `{ query: string, params: any[] }`:
```ts
console.log(sql`select * from users`.toSQL());
console.log(sql`select * from users where id = ${10}`.toSQL());
```

```shell
{
  query: "select * from users",
  params: [],
}
{
  query: "select * from users where id = $1",
  params: [ 10 ],
}
```
