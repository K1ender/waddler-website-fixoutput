# Waddler \<\> OP SQLite
According to the **[official github page](https://github.com/OP-Engineering/op-sqlite)**, 
OP-SQLite embeds the latest version of SQLite and provides a low-level API to execute SQL queries.

<Npm>
waddler @op-engineering/op-sqlite
</Npm>

```ts
import { waddler } from "waddler/op-sqlite";
import { open } from '@op-engineering/op-sqlite';

const opsqlite = open({
  name: 'myDB',
});
const sql = waddler({ client: opsqlite });

const result = await sql`select 1;`.all();
```
