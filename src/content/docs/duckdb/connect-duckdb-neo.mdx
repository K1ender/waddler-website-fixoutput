# Waddler \<\> DuckDB
<Prerequisites>
- duckdb-neo [basics](https://duckdb.org/docs/stable/clients/node_neo/overview/)
</Prerequisites>

Waddler has native support for DuckDB connections with the `duckdb-neo` driver.

#### Step 1 - Install packages
<Npm>
waddler @duckdb/node-api
</Npm>

#### Step 2 - Initialize the driver and make a query
<CodeTabs items={["duckdb-neo", "duckdb-neo in memory"]}>
```typescript copy
import { waddler } from 'waddler/duckdb-neo';

sql = waddler({ url: process.env.DATABASE_URL!, max: 10, accessMode: 'read_write' });
```
```typescript copy
import { waddler } from 'waddler/duckdb-neo';

sql = waddler({ url: ':memory:', max: 10, accessMode: 'read_write' });
```
</CodeTabs>
