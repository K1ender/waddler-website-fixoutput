# Waddler \<\> Gel
<Prerequisites>
- Database [connection basics](/docs/connect-overview) with Waddler
- gel-js [basics](https://github.com/geldata/gel-js)
</Prerequisites>

Waddler has native support for Gel connections with the `gel-js` client.

#### Step 1 - Install packages
<Npm>
waddler gel
</Npm>

#### Step 2 - Initialize the driver and make a query
<CodeTabs items={["gel", "gel with config"]}>
```typescript copy
// Make sure to install the 'gel' package 
import { waddler } from 'waddler/gel';

const sql = waddler(process.env.DATABASE_URL);
 
const result = await sql`select 1;`;
```
```typescript copy
// Make sure to install the 'gel' package
import { waddler } from "waddler/gel";

// You can specify any property from the gel connection options
const sql = waddler({
  connection: {
    dsn: process.env.DATABASE_URL,
    tlsSecurity: "default",
  },
});

const result = await sql`select 1;`;
```
</CodeTabs>

If you need to provide your existing driver:

```typescript copy
// Make sure to install the 'gel' package 
import { waddler } from "waddler/gel";
import { createClient } from "gel";

const gelClient = createClient();
const sql = waddler({ client: gelClient });

const result = await sql`select 1;`;
```
